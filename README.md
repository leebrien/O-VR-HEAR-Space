# O-VR HEAR
**O-VR HEAR** is a Unity-based Virtual Reality research application designed to simulate virtual sound localization with 3 different localization tasks under two audio cue conditions: Generic (GC) and Personalized (PC). It aims to evaluate the impact of personalized vs. generic spatial audio cues on sound localization performance. The project also supports integrated VR questionnaires and real-time cue switching for within-subject experimental designs.

This system is integrated with Hearsona, providing the AI-generated Personalized audio cues through a web endpoint

## Features

### Spatial Audio Localization Tasks
The system implements three research-based localization tasks:

1. **Task 1 – Pointing Task**  
   Point toward the direction of a spatialized audio source.

2. **Task 2 – Locate & Grab**  
   Navigate and physically grab an invisible sounding target.

3. **Task 3 – Locate & Grab with Distractors**  
   Same as Task 2 but with competing ambient audio sources.

All tasks support both **generic** and **personalized** cues.

## Getting Started

### Requirements
- Unity 2021.3 LTS or later  
- XR Interaction Toolkit  
- OpenXR Plugin  
- VR headset (Meta Quest recommended)

### Installation
1. Clone the repository (Git LFS Required)
2. Open the project in Unity  
3. Install required packages  
4. Configure XR settings for your headset  
5. Press Play or build to device
